# Vinny Pham Portfolio

Live Site: https://vinny-pham.vercel.app/

---

## About

This is my personal portfolio website built to showcase my projects, skills, and experience as a Computer Science student at California State University, Fullerton.

The site features a modern, responsive design with smooth animations and a dark theme, inspired by clean developer portfolios.

---

## Tech Stack

- **Frontend:** React, Next.js (App Router)
- **Styling:** CSS
- **Animations:** Framer Motion
- **Deployment:** Vercel

---

## ✨ Features

- Responsive, modern UI
- Dark theme design
- Smooth scrolling navigation
- Animated sections using Framer Motion
- Project showcase with interactive cards
